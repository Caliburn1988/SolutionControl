# SolutionControl

这是一个基于 Qt 的解决方案控制界面示例程序。

## 功能特点

- 加载和保存解决方案
- 密码保护
- 工作区锁定/解锁
- 多语言支持（中文/英文）
- 自适应界面布局

## 开发环境要求

- Qt 5.15.2
- Visual Studio 2019
- Windows 10 或更高版本

## 构建说明

1. 确保已安装 Qt 5.15.2 和 Visual Studio 2019
2. 使用 Qt Creator 打开 `SolutionControl.pro`
3. 选择 MSVC 2019 64-bit 编译器
4. 点击构建按钮进行编译

## 自动构建

本项目使用 GitHub Actions 进行自动构建，每次提交都会触发构建流程。构建结果可在 Actions 页面查看。
